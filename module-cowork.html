Module 3 · Workshop

Working with Cowork.

Cowork is the desktop mode that lets Claude touch files, run skills, and finish multi-step tasks on your machine. We’ll review Anthropic’s Cowork 101, run an advanced example, and wrap with office hours and the roadmap.

Cowork 101 recap

Four things to remember.

Files

Your folder, your data

Cowork works inside a folder you pick. Drag-drop to add files, and Claude can read and write inside that folder only.

Skills

Pluggable capabilities

Skills are reusable instructions Claude can run — for documents, PDFs, spreadsheets, contracts, and more.

Tools

Connectors & the shell

Cowork can call MCP connectors (when approved) and a sandboxed shell to run small scripts.

Memory

Reusable context

Project instructions and a memory file let you keep institutional knowledge in one place across many sessions.
