Our goal is to help create efficiency by automating tedious work.

Not replace your judgment. Not write the parts that need you. The parts that take time and don’t take thought.

Slide 4 · Data rules

Same rules as Sandbox AI.

  • OK to use: confidential Cornell data used in research administration — protocols, awards, proposals, IP.
  • Not allowed: HIPAA data (we shouldn’t encounter it) and export-controlled or government-restricted information.
  • Claude 3P is a different front end pointing at the same infrastructure as Sandbox AI — the LiteLLM gateway to the cloud models Cornell already has data-protection contracts with.
  • Web search, when used, runs through Gemini’s enterprise grounding — your prompt is not logged.

If you can put it in Sandbox AI, you can put it in Claude 3P. Same gateway, same contracted infrastructure.

Slide 5 · How AI works

Probabilistic, not deterministic.

An LLM builds its answer one token at a time, picking among plausible next words by probability. There is no fixed right answer — only what’s most likely given what came before.

That’s why two runs of the same prompt can come back worded differently, and occasionally with different substance. It’s a feature, not a bug — but it’s why you read every answer critically and spot-check the things that matter.

Slide 6 · The 3-part prompt

Set the stage. Give the task. Specify the output.

1 · Stage

Set the stage

Who you are, what you’re doing, what success looks like. Attach reference files if useful.

2 · Task

Give the task

One clear, specific ask. Break compound work into numbered steps.

3 · Output

Specify the output

Format, length, audience, tone. Paste an example if you have one.

Slide 8 · The interface

What you see when you open Claude 3P.

  • Chat box. Where you type your prompt and drop in files.
  • Model picker (bottom). Sonnet for most work, Opus for hard reasoning, Haiku for quick lookups.
  • File drop. Drag and drop — PDFs, images, Word, Excel, CSV, code files all work.
  • Sidebar. Your previous chats, stored on your local machine until you delete them.
