Today’s exercise

Test against data you already know cold.

Hands-on · ~25 min

Validate Claude against an answer you already know

Bring one dataset you understand — an annual award report, a recent portfolio review, a closed proposal cycle, or a compliance dashboard. Then run the loop: ask, compare, refine, or find the limit.

I'm a research administrator at Cornell. Here is [DATASET DESCRIPTION
+ FILE] — a [dataset I already know cold].

I'm going to ask you a question I already know the answer to, and we'll
compare. This is a validation pass before I trust you with new data.

Question: [your question]

Show your reasoning briefly. Tell me which fields you used and how you
combined them. If anything is ambiguous, ask before answering.

The takeaway to land: before you trust AI with new data, validate on data you already know. That’s Deployment Diligence in practice.
