# Feedback Log

Customer / cohort feedback collected during and after training sessions, kept internally and used to shape content over time. Most recent at the top.

**Use this for:** comments, questions, and reactions from workshop participants, office-hours visitors, email replies, and anyone else using the training. The goal is to spot patterns — what confuses people, what they ask for, what they wish was different — and feed them back into the slides, exercises, FAQ, and site content.

**Suggested entry format:**

```
### [date] — [cohort or source]
- Verbatim quote or close paraphrase, with light context if needed.
- Tag with [Confusing], [Request], [Praise], [Bug], [Idea] if useful.
- Note any follow-up action taken (or "—" if none yet).
```

**What this file is *not*:** an internal decision log. Design choices, rationale, deferred work, and open questions about how to build the content live in memory or working notes, not here.
